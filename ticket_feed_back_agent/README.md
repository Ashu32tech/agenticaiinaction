# Agentic AI Hybrid System (CrewAI + Streamlit)

## Run Steps

1. Install dependencies
pip install -r requirements.txt

2. Run Streamlit UI
streamlit run app.py

## Features
- Streamlit dashboard
- Multi-agent pipeline (CrewAI)
- Traceability (trace_id)
- Logging (processing_log.csv)
- Test cases included
